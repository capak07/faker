# Faker::God

Available since version 1.9.0.

```ruby
# Random Mythology
Faker::God.mythology #=> "Roman"

# Random Greek God Name
Faker::God.greek_god_name #=> "Athena"

# Random Roman God Name
Faker::God.roman_god_name #=> "Apollo"

# Random Norse God Name
Faker::God.norse_god_name #=> "Loki"

# Random Hindu God Name
Faker::God.hindu_god_name #=> "Indra"

# Random Mayan God Name
Faker::God.mayan_god_name #=> "Athena"

# Random Chinese God Name
Faker::God.chinese_god_name #=> "Dragon King"

# Random Egyptian God Name
Faker::God.egyptian_god_name #=> "Khonshu"
